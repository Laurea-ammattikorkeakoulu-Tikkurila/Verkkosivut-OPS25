Workshop 1: Introduction to HTML

In this workshop, you will learn the basics of HTML, the standard markup language for creating web pages. HTML elements are the building blocks of HTML pages.

Task 1: Create a Basic HTML Page

Create a simple HTML page with a title, heading, and paragraph. Use the following structure: Add a heading for your name, include a paragraph describing yourself

<!DOCTYPE html>
    <html>
    <head>
        <title>My First HTML Page</title>
    </head>
    <body>
        <h1>Welcome to my Page</h1>
        <p>This is a paragraph.</p>
    </body>
    </html>

Task 2: Insert an image of yourself with appropriate alt text

Include your own picture in your HTML page using the <img> tag. Example:

<img src="path/to/image.jpg" alt="Description of image">

Task 3: Create a List

Add a list (ordered or unordered) of your hobbies. Example:

<h2>Ordered List</h2>
    <ol>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
    </ol>
    
    <h2>Unordered List</h2>
    <ul>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
    </ul>

Task 4: Add a Link

Add a hyperlink to your favorite Web page <a> tag. Example:

<a href="https://www.laurea.fi">Visit Laurea</a>

Bonus Task: Create a Table

Create a table with at least 3 rows and 2 columns. Include a header row and some sample data. Example:

<table>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
        <tr>
            <td>Row 1, Cell 1</td>
            <td>Row 1, Cell 2</td>
        </tr>
        <tr>
            <td>Row 2, Cell 1</td>
            <td>Row 2, Cell 2</td>
        </tr>
    </table>

Workshop 2: Introduction to CSS

In this workshop, you will learn the basics of CSS, the language used to style HTML elements.

Task 1: Change Text Color

Change the color of the text in your HTML page using the color property. Example:

p {
    color: blue;
}

Task 2: Add Background Color

Add a background color to your HTML page using the background-color property. Example:

body {
    background-color: lightgray;
}

Task 3: Change Font Size

Change the font size of the text in your HTML page using the font-size property. Example: 

h1 {
    font-size: 2em;
}

Task 4: Add Padding and Margin

Add padding and margin to elements in your HTML page using the padding and margin properties. Example: 

div {
    padding: 20px;
    margin: 10px;
}

Bonus Task: Create a CSS Grid Layout

Create a simple grid layout using CSS Grid. Learn more about the CSS grid layout in MDN Web Docs. Example: 

.grid-container {
    display: grid;
    grid-template-columns: auto auto auto;
    gap: 10px;
}

.grid-item {
    background-color: #ddd;
    padding: 20px;
    text-align: center;
}

Workshop 3: Web page layout

In this workshop, you will learn about different techniques for laying out a web page using HTML and CSS.

Task 1: Create a Header

Create a header for your web page that includes a logo and a navigation menu. Use the following structure:

<header>
            <img src="path/to/logo.png" alt="Logo" style="height: 50px;">
            <nav>
                <ul>
                    <li><a href="#home">Home</a></li>
                    <li><a href="#about">About</a></li>
                    <li><a href="#contact">Contact</a></li>
                </ul>
            </nav>
        </header>

Task 2: Create a Two-Column Layout

Create a two-column layout for your web page using CSS. One column should be for the main content and the other for a sidebar. Example:

.container {
            display: flex;
        }
        
        .main-content {
            flex: 3;
            padding: 20px;
        }
        
        .sidebar {
            flex: 1;
            padding: 20px;
            background-color: #f4f4f4;
        }

Task 3: Create a Footer

Create a footer for your web page that includes some basic information and links. Use the following structure:

<footer>
            <p>© 2025 Your Name. All rights reserved.</p>
            <nav>
                <ul>
                    <li><a href="#privacy">Privacy Policy</a></li>
                    <li><a href="#terms">Terms of Service</a></li>
                </ul>
            </nav>
        </footer>

Task 4: Add a Hero Section

Add a hero section to your web page that includes a large background image and some text. Example:

.hero {
            background-image: url('path/to/image.jpg');
            background-size: cover;
            background-position: center;
            color: white;
            text-align: center;
            padding: 100px 20px;
        }
        
        .hero h1 {
            font-size: 3em;
        }
        
        .hero p {
            font-size: 1.5em;
        }

Bonus Task: Create a Responsive Layout

Create a responsive layout for your web page that adjusts to different screen sizes using media queries. Example:

@media (max-width: 768px) {
            .container {
                flex-direction: column;
            }
        
            .main-content, .sidebar {
                flex: 1;
            }
        }

Workshop 4: Advanced CSS

In this workshop, you will learn advanced CSS techniques to enhance your web pages.

Task 1: CSS Transitions

Add a transition effect to an element when it changes state, such as on hover. Example:

button {
            background-color: #4CAF50;
            color: white;
            padding: 15px 32px;
            text-align: center;
            text-decoration: none;
            display: inline-block;
            font-size: 16px;
            margin: 4px 2px;
            cursor: pointer;
            transition: background-color 0.3s ease;
        }
        
        button:hover {
            background-color: #45a049;
        }

Task 2: CSS Animations

Create a simple animation using CSS. Example:

@keyframes example {
            from {background-color: red;}
            to {background-color: yellow;}
        }
        
        div {
            width: 100px;
            height: 100px;
            background-color: red;
            animation-name: example;
            animation-duration: 4s;
        }

Task 3: CSS Flexbox

Use Flexbox to create a flexible layout. Example:

.flex-container {
            display: flex;
            justify-content: space-between;
        }
        
        .flex-item {
            background-color: #f1f1f1;
            padding: 10px;
            margin: 5px;
            text-align: center;
        }

Task 4: CSS Grid

Use CSS Grid to create a grid layout. Example:

.grid-container {
            display: grid;
            grid-template-columns: auto auto auto;
            gap: 10px;
        }
        
        .grid-item {
            background-color: #ddd;
            padding: 20px;
            text-align: center;
        }

Bonus Task: CSS Variables

Use CSS variables to manage colors and other styles more efficiently. Example:

:root {
            --main-bg-color: lightblue;
            --main-text-color: darkblue;
        }
        
        body {
            background-color: var(--main-bg-color);
            color: var(--main-text-color);
        }

Workshop 5: Responsive Design

In this workshop, you will learn about responsive design techniques to make your web pages look good on all devices.

Task 1: Create a Responsive Layout Using Div and Class

Create a responsive layout using `div` elements and classes. Example:

<div class="container">
            <div class="header">Header</div>
            <div class="content">Main Content</div>
            <div class="sidebar">Sidebar</div>
            <div class="footer">Footer</div>
        </div>

CSS:

.container {
            display: flex;
            flex-wrap: wrap;
        }
        
        .header, .footer {
            background-color: #f1f1f1;
            width: 100%;
            padding: 20px;
            text-align: center;
        }
        
        .content {
            background-color: #ddd;
            flex: 3;
            padding: 20px;
        }
        
        .sidebar {
            background-color: #ccc;
            flex: 1;
            padding: 20px;
        }

Task 2: Use the Box Model

Apply the box model to style elements with padding, border, and margin. Example:

.box {
            width: 300px;
            padding: 20px;
            border: 5px solid black;
            margin: 20px;
            background-color: lightblue;
        }

HTML:

<div class="box">This is a box using the box model.</div>

Task 3: Position Elements

Use CSS positioning to place elements on the page. Example:

.relative-box {
            position: relative;
            width: 200px;
            height: 200px;
            background-color: lightgreen;
        }
        
        .absolute-box {
            position: absolute;
            top: 50px;
            left: 50px;
            width: 100px;
            height: 100px;
            background-color: lightcoral;
        }

HTML:

<div class="relative-box">
            Relative Box
            <div class="absolute-box">Absolute Box</div>
        </div>

Task 4: Media Queries

Use media queries to create a responsive design that adjusts to different screen sizes. Example:

@media (max-width: 768px) {
            .container {
                flex-direction: column;
            }
        
            .content, .sidebar {
                width: 100%;
            }
        }

Bonus Task: Responsive Navigation Menu

Create a responsive navigation menu that collapses into a hamburger menu on smaller screens. Example:

.nav {
            display: flex;
            justify-content: space-between;
            background-color: #333;
            padding: 10px;
        }
        
        .nav a {
            color: white;
            padding: 14px 20px;
            text-decoration: none;
            text-align: center;
        }
        
        .nav a:hover {
            background-color: #ddd;
            color: black;
        }
        
        .hamburger {
            display: none;
            font-size: 30px;
            cursor: pointer;
        }
        
        @media (max-width: 768px) {
            .nav a {
                display: none;
            }
        
            .nav a.icon {
                display: block;
            }
        
            .nav.responsive {
                position: relative;
            }
        
            .nav.responsive a {
                display: block;
                text-align: left;
            }
        
            .hamburger {
                display: block;
            }
        }

HTML:

<div class="nav" id="myNav">
            <a href="#home">Home</a>
            <a href="#about">About</a>
            <a href="#services">Services</a>
            <a href="#contact">Contact</a>
            <a href="javascript:void(0);" class="icon" onclick="toggleNav()">☰</a>
        </div>
        
        <script>
        function toggleNav() {
            var x = document.getElementById("myNav");
            if (x.className === "nav") {
                x.className += " responsive";
            } else {
                x.className = "nav";
            }
        }
        </script>
