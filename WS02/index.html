Workshop 2: CSS Fundamentals

In this workshop, you will learn the basics of CSS, the language used to style HTML elements.

Task 1: Change Text Color

Change the color of the text in your HTML page using the color property. Example: 

p {
        color: blue;
      }

Task 2: Add Background Color

Add a background color to your HTML page using the background-color property. Example: 

body {
        background-color: lightgray;
      }

Task 3: Change Font Size

Change the font size of the text in your HTML page using the font-size property. Example: 

h1 {
        font-size: 2em;
      }

Task 4: Add Padding and Margin

Add padding and margin to elements in your HTML page using the padding and margin properties. Example: 

div {
        padding: 20px;
        margin: 10px;
      }

Task 5: Format the Table and Form

Format the table and form you created in the previous workshop using CSS. Add styles to make them visually appealing. Example: 

table {
        width: 100%;
        border-collapse: collapse;
      }
      
      th, td {
        border: 1px solid #ddd;
        padding: 8px;
        text-align: left;
      }
      
      th {
        background-color: #f2f2f2;
      }
      
      form {
        display: flex;
        flex-direction: column;
        gap: 10px;
      }
      
      label {
        font-weight: bold;
      }
      
      input[type="text"], input[type="email"] {
        padding: 8px;
        border: 1px solid #ddd;
        border-radius: 4px;
      }
      
      input[type="submit"] {
        background-color: #4CAF50;
        color: white;
        padding: 10px;
        border: none;
        border-radius: 4px;
        cursor: pointer;
      }
      
      input[type="submit"]:hover {
        background-color: #45a049;
      }

Task 6: Validate Your HTML Code

Validate your CSS code using the W3C Web Validator extension in Visual Studio Code. Follow these steps:

  1. Install the W3C Web Validator extension in Visual Studio Code.
  2. Open your CSS file in Visual Studio Code.
  3. Use the W3C Web Validator extension to validate your code.
  4. Fix any errors or warnings that are reported.
