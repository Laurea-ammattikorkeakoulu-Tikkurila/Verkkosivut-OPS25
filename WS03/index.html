Workshop 3: Advanced CSS

In this workshop, you will learn advanced CSS techniques including Flexbox, Grid Layout, and Media Queries.

Task 1: Flexbox Layout

Create a Flexbox layout with three columns. Each column should have a different background color. Use the following structure: 

<div class="flex-container">
        <div class="flex-item">Column 1</div>
        <div class="flex-item">Column 2</div>
        <div class="flex-item">Column 3</div>
      </div>

CSS:

.flex-container {
        display: flex;
        justify-content: space-around;
      }
      
      .flex-item {
        background-color: lightblue;
        padding: 20px;
        margin: 10px;
        flex: 1;
      }

Task 2: Grid Layout

Create a Grid Layout with four columns and two rows. Each cell should have a different background color. Use the following structure: 

<div class="grid-container">
        <div class="grid-item">1</div>
        <div class="grid-item">2</div>
        <div class="grid-item">3</div>
        <div class="grid-item">4</div>
        <div class="grid-item">5</div>
        <div class="grid-item">6</div>
        <div class="grid-item">7</div>
        <div class="grid-item">8</div>
      </div>

CSS:

.grid-container {
        display: grid;
        grid-template-columns: repeat(4, 1fr);
        grid-gap: 10px;
      }
      
      .grid-item {
        background-color: lightgreen;
        padding: 20px;
        text-align: center;
      }

Task 3: Media Queries

Create a responsive layout that changes the background color of a div based on the screen width. Use the following structure: 

<div class="responsive-div">Resize the window to see the effect</div>

CSS:

.responsive-div {
        padding: 20px;
        text-align: center;
      }
      
      @media (max-width: 600px) {
        .responsive-div {
          background-color: lightcoral;
        }
      }
      
      @media (min-width: 601px) {
        .responsive-div {
          background-color: lightseagreen;
        }
      }

Task 4: Flexbox Alignment

Create a Flexbox layout with items aligned to the center both vertically and horizontally. Use the following structure: 

<div class="flex-center-container">
        <div class="flex-center-item">Centered Item</div>
      </div>

CSS:

.flex-center-container {
        display: flex;
        justify-content: center;
        align-items: center;
        height: 200px;
        background-color: lightgray;
      }
      
      .flex-center-item {
        background-color: lightblue;
        padding: 20px;
      }

Task 5: Grid Layout with Media Queries

Create a Grid Layout that changes the number of columns based on the screen width. Use the following structure: 

<div class="responsive-grid-container">
        <div class="responsive-grid-item">1</div>
        <div class="responsive-grid-item">2</div>
        <div class="responsive-grid-item">3</div>
        <div class="responsive-grid-item">4</div>
        <div class="responsive-grid-item">5</div>
        <div class="responsive-grid-item">6</div>
      </div>

CSS:

.responsive-grid-container {
        display: grid;
        grid-gap: 10px;
      }
      
      @media (max-width: 600px) {
        .responsive-grid-container {
          grid-template-columns: repeat(2, 1fr);
        }
      }
      
      @media (min-width: 601px) {
        .responsive-grid-container {
          grid-template-columns: repeat(3, 1fr);
        }
      }
      
      .responsive-grid-item {
        background-color: lightpink;
        padding: 20px;
        text-align: center;
      }
